hexo 官网： https://hexo.io/

hexo 中文安装文档： https://hexo.io/zh-cn/docs/index.html

  1. 安装 node.js： https://nodejs.org/en/
  2. 安装 git： http://git-scm.com/ ，并配置好环境变量

  3. 安装 hexo

  4. 配置_config.yml
